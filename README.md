# Treasury UI

BengoBox treasury management UI. Transactions, settlements, gateways, platform admin.

## Dev

```bash
pnpm install
pnpm dev
```

## Deploy

Deployed at `https://books.codevertexitsolutions.com`. CI on push to `main`.

## Push from this folder (own repo)

```bash
# From: finance-service/treasury-ui
git remote add origin https://github.com/YOUR_ORG/treasury-ui.git
git branch -M main
git push -u origin main
```
